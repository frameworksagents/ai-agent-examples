# Pipeline multi-agents

Un pipeline multi-agents consiste à faire intervenir plusieurs agents spécialisés dans un même processus.

## Exemple

1. agent recherche
2. agent synthèse
3. agent vérification
4. agent rédaction

Chaque agent intervient sur une étape spécifique.

## Avantages

- meilleure spécialisation
- qualité de sortie améliorée
- architecture modulaire

## Cas d’usage

- génération de contenu
- analyse de documents
- production de rapports
