# Agent utilisant des outils externes

Les agents IA peuvent utiliser des outils pour interagir avec des systèmes externes.

Ces outils peuvent être :

- API
- bases de données
- moteurs de recherche
- services internes

## Exemple

Un agent peut :

1. recevoir une demande
2. appeler une API
3. analyser la réponse
4. produire un résultat

## Intérêt

L’utilisation d’outils permet aux agents de :

- accéder à des données réelles
- effectuer des actions
- interagir avec des systèmes existants
