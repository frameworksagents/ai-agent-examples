# Agent de recherche

Un agent de recherche permet de collecter des informations sur un sujet donné, d’identifier les sources pertinentes et de produire une synthèse structurée.

## Objectif

Transformer une requête utilisateur en un ensemble d’informations synthétisées.

## Étapes typiques

1. analyser la requête
2. effectuer une recherche
3. collecter les sources
4. extraire les points clés
5. produire une synthèse

## Utilisation

Ce type d’agent est souvent utilisé pour :

- veille technologique
- recherche documentaire
- analyse de marché
- préparation d’articles
