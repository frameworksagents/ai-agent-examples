# AI Agent Examples

Ce dépôt contient des **exemples pratiques d’agents IA**, des **patterns d’orchestration** et des **prompts réutilisables** pour construire des systèmes d’automatisation basés sur des agents.

Site principal :

https://frameworksagents.com

## Contenu du dépôt

### Exemples d’agents

- agent de recherche
- agent de tri de tickets support
- pipeline multi-agents
- agent utilisant des outils externes

### Prompts

- prompt agent recherche
- prompt agent synthèse
- prompt agent relecture

### Patterns d’architecture

- planner executor
- router agent
- review and revise

## Objectif

Ces exemples permettent de comprendre comment concevoir des **architectures d’agents IA simples et robustes**.

Les guides complets sont disponibles sur :
